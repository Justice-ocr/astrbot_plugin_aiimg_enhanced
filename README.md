# AI绘图站

> 基于 [astrbot_plugin_gitee_aiimg](https://github.com/muyouzhi6/astrbot_plugin_gitee_aiimg) 增强版

多服务商文生图 / 改图 / 多人设自拍 / 视频生成插件，附可视化配置页面。

---

## 功能特性

### 🎨 生图能力
- **文生图** `/aiimg` `/生图` `/画图` — 支持 12+ 服务商
- **改图 / 图生图** `/aiedit` `/改图` — 多后端自动兜底
- **自拍参考照** `/自拍` — 基于参考图生成一致人脸自拍
- **视频生成** `/视频` — 异步任务，支持首尾帧
- **批量出图** `/!!批量N` — 并发执行，LLM 规划提示词

### 🎭 多人设管理
- 每套人设独立名称、外貌描述、参考图组
- `/人设` — 查看列表
- `/切换人设 <序号/ID/名称>` — 即时切换并持久化

### ⚙️ 可视化配置页面
在 AstrBot WebUI 的插件详情页打开，支持：

| 标签页 | 内容 |
|--------|------|
| 功能开关 | 各功能启用/禁用、服务商链路配置、输出尺寸、批量并发 |
| 服务商 | 增删改 17+ 种服务商，字段随类型动态变化 |
| 预设指令 | 文生图 / 改图 / 视频 预设管理 |
| 人设管理 | 新建/编辑人设，支持上传参考图（多选） |
| 状态文案 | 自定义等待/失败提示，支持 `{persona_name}` `{error}` 等占位符 |
| 高级配置 | 防抖、并发上限、缓存、网络安全参数 |

### 💬 自定义状态文案
```
🎨 收到灵感，正在绘制...          ← 可自定义
📸 正在为「{persona_name}」生成自拍，请稍候...
💥 绘制失败: {error}
```

## 支持的服务商

| 类型 | 服务商 |
|------|--------|
| 图像生成 | OpenAI Images、Gemini 原生、Gitee Images、Grok Images、Vertex AI 匿名、Flow2API、魔搭 等 |
| 图像编辑 | Gitee 异步改图、OpenAI Images Edit、Gemini Edit 等 |
| 视频生成 | Grok 视频、Flow2API 视频 |
| 其他 | 即梦（豆包）、Grok2API |

## 安装

在 AstrBot 插件市场搜索 **AI绘图站**，或手动上传 zip 安装包。

### 配置

安装后在插件详情页的 **Settings** 标签页配置服务商和功能链路，保存后立即生效，无需重启。

## 指令列表

| 指令 | 说明 |
|------|------|
| `/aiimg <提示词>` | 文生图（别名：`/生图` `/画图` `/绘图` `/出图`） |
| `/文生图 <提示词>` | 支持预设触发的文生图 |
| `/aiedit <提示词>` | 改图（别名：`/改图` `/图生图` `/修图`） |
| `/<预设名> [提示词]` | 预设改图，如 `/手办化` |
| `/自拍 [提示词]` | 用当前人设生成自拍 |
| `/自拍参考 [图片]` | 设置/查看自拍参考图 |
| `/视频 <提示词>` | 生成视频 |
| `/人设` | 查看人设列表 |
| `/切换人设 <序号/ID/名称>` | 切换当前人设 |
| `/!!批量N <提示词>` | 批量出图 |
| `/预设列表` | 查看改图预设 |
| `/文生图预设列表` | 查看文生图预设 |
| `/视频预设列表` | 查看视频预设 |
| `/改图帮助` | 显示改图帮助 |
| `/重发图片` | 重新发送上一张图 |

所有生图指令均支持 `@服务商ID` 前缀临时指定服务商，如 `/aiimg @openai 一只猫`。
